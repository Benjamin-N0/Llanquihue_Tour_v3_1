## Descripción general del sistema

Esta aplicación fue desarrollada en Java y tiene como objetivo gestionar información de rutas turísticas. Además, permite cargar datos desde archivos, almacenarlos en colecciones y realizar búsquedas de manera sencilla.


## 🧱 Estructura general del proyecto
```plaintext
📁 src/
├── cl.llanquihuetour.app/         # Clase principal con el método main
├── cl.llanquihuetour.model/       # Clases de dominio (Persona, Guia, RutaTuristica)
├── cl.llanquihuetour.service/     # Clase de servicio (buscar rutas, filtrar)
├── cl.llanquihuetour.util/        # Clase de gestor de datos
````
## Ejecución

1. Clonar el repositorio:

```bash
git clone https://github.com/Benjamin-N0/llanquihue-tour.git
```

2. Abrir el proyecto en NetBeans.

3. Ejecutar la clase:

```text
cl.llanquihuetour.app/Main.java
```
